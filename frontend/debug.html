💡 How It Works
Debug Console (This Page)
Connects directly to a local agent via ws://localhost:8080/ws.
No authentication needed. For local development and debugging only.
Production (React Frontend)
1. Cognito Identity Pool provides AWS credentials
2. @aws-sdk/signature-v4 generates SigV4 presigned URL
3. Browser connects via wss:// with auth in query params
Usage
1. Start agent: cd agent && python strands_agent.py
2. Open this page in browser
3. Click Connect, then Click to Speak
